More About Request Header!

  • Whenever browser is sending a request to servlet that request is having three things.

    1.Request Line --> Contains
    a. Method Type (GET OR POST)
    b. URI -> Address of the destination.
    c. HttpVersion.(HTTP/0.9, HTTP/1.0/HTTP/1.1)

    2. Request Header--> Passing name : value
    Example : image/gif, image/jpeg.

    3.Request Body.
    if request is Get, then there will be no request body.
    if request is Post, then all the request parameter will be under request body.
