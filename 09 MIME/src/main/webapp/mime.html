MIME (Multipurpose Internet mail extension)

By Using MIME, We are represent the type of the data which is transfer through the Internet.
  • MIME is generic type to any language.
    In java, we can represent by using response sponse.setContentType("Type/subtype")

    To represent html content->response.setContentType("text/html")
    To represent plain text document--> response.setContentType("text/plain")

    To transfer CSS-->response.setContentType("text/css")
    To transfer XML (if our servlet want to sent xml output to the browser) -->response.setContentType("text/xml")

    To send image as output to browser to transfer gif from one place to other over internet -->response.setContentType("image/gif")
    To transfer image of jpeg format -->response.setContentType("image/jpeg")

    To transfer png image -->response.setContentType("image/png")
    To transfer windows bitmap image -->response.setContentType("image/x-xbitmap")

    To transfer microsoft word document -->response.setContentType("application/ms-word")
    To transfer microsoft excel spreadsheet -->response.setContentType("application/vnd.ms-excel")

    To transfer of portable document format->response.setContentType("application/pdf")
    To transfer postscript file (unix-related) -->response.setContentType("application/postscript")

    To transfer zip file->response.setContentType("application/zip")
    To transfer of gzip file ->response.setContentType("application/x-gzip")

    To transfer jar file->response.setContentType("application/x-java-archieve")
    To transfer binary data over network ->response.setContentType("application/octel-stream")

    To transfer of midi file sound file->response.setContentType("audio/x-midi")
    To transfer of mp3 sound file ->response.setContentType("audio/mpeg")

    To transfer of wav sound file->response.setContentType("audio/wav")
    To transfer of quicktime vedio file ->response.setContentType("video/quicktime")
